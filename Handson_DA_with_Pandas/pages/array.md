- concept
  An array is a fundamental data structure that stores a collection of items (like numbers or text) of the same type in a single variable, organized in an ordered list with each item accessed by a numerical index, usually starting from 0.
	- Key Characteristics
	- **[Collection](https://www.google.com/search?q=Collection&oq=what+is+array&gs_lcrp=EgZjaHJvbWUyCQgAEEUYORiABDIHCAEQABiABDIHCAIQABiABDIHCAMQABiABDIHCAQQABiABDIHCAUQABiABDIHCAYQABiABDIHCAcQABiABDIHCAgQABiABDIHCAkQABiABNIBCDQxMjFqMGo3qAIAsAIA&sourceid=chrome&ie=UTF-8&ved=2ahUKEwjs0uWIjLmRAxWYKFkFHcqaDkUQgK4QegYIAQgAEAs) of similar items**: Stores multiple pieces of data (elements) under one name, like an array of "scores" or "names".
	- **Ordered**: Elements have a specific position, so `array[0]` is the first item, `array[1]` is the second, and so on.
	- **Contiguous Memory**: In many languages, elements are stored next to each other in memory for fast access.
	- **Index**: A number used to identify and retrieve an element (e.g., `myArray[2]`).
	- **Length**: The total number of elements in the array
